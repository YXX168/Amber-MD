# Amber MD 🦐

> 极简玻璃态 Markdown 阅读器

一款专注于 **美学与体验** 的 Markdown 文档阅读器，采用 glassmorphism（玻璃态）设计风格，支持 5 种霓虹动画主题。

---

## ✨ 特色功能

- 🎨 **5种霓虹主题** — 午夜/极光/森林/海洋/樱花，每款都有独特的渐变动画背景
- 📱 **玻璃态UI** — 毛玻璃卡片、透明浮动按钮，现代感十足
- ☁️ **WebDAV云同步** — 连接坚果云/NextCloud等，随时随地访问文档
- 🌙 **沉浸式阅读** — 自动隐藏顶栏、返回顶部按钮、搜索高亮
- ✏️ **编辑模式** — 快速修改文档内容并保存
- 💾 **本地文件支持** — 打开手机/平板上的 .md / .txt / .json 文件

## 🔧 WebDAV 配置

支持坚果云、NextCloud、群晖 NAS 等标准 WebDAV 服务：

1. 打开「网络存储」页面
2. 输入 WebDAV 地址（如 `https://dav.jianguoyun.com/dav/`）
3. 填写账号密码
4. 点击连接，浏览云端文档
